# HireAIlytics --- AI Resume Analyzer

HireAIlytics is an AI-powered resume analysis platform that evaluates
resumes against job descriptions and provides actionable feedback, ATS
compatibility scores, and improvement suggestions.

The goal is to help candidates optimize their resumes for Applicant
Tracking Systems (ATS) and increase their chances of getting shortlisted
by recruiters.

------------------------------------------------------------------------

## Features

-   **AI Resume Analysis**
    -   Upload a resume and receive AI-generated feedback.
-   **ATS Compatibility Score**
    -   Measures how well the resume performs in Applicant Tracking
        Systems.
-   **Detailed Resume Insights**
    -   Tone & Style evaluation
    -   Content quality assessment
    -   Resume structure analysis
    -   Skills evaluation
-   **Actionable Suggestions**
    -   Personalized improvement tips generated by AI.
-   **Resume Preview**
    -   Displays uploaded resume with analyzed results.
-   **Secure File Storage**
    -   Uses Puter filesystem API for storing files.
-   **Modern UI**
    -   Built with TailwindCSS for a clean, responsive interface.

## How It Works

1.  User uploads a resume (PDF).
2.  The system converts the PDF into an image preview.
3.  Resume is uploaded to the Puter File System.
4.  AI analyzes the resume against the job description.
5.  Feedback is returned in structured JSON format.
6.  Results are stored in Puter KV storage.
7.  Resume review dashboard displays the analysis.
